# Security Policy

## Reporting A Vulnerability

Please report suspected vulnerabilities privately by emailing
support@evokoa.com.

Include:

- Affected package version.
- A clear description of the issue.
- Reproduction steps or proof of concept, if available.
- Any known impact or exposure.

Do not open a public GitHub issue for security vulnerabilities. We will review
reports and coordinate remediation before public disclosure when appropriate.

## Supported Versions

Security fixes target the latest released version of the Polygres Python SDK.
The SDK is currently in alpha, so users should upgrade promptly when a security
release is published.
